Greetings, traveler of the World Wide Web!

This project is all about providing you with a simple way to access data about the regions available in Microsoft Azure.

You can view the latest Azure regions data by clicking the link below:

Azure Regions Data

What does this project do?

Using the power of the Azure CLI, we generate a JSON file that contains all the Azure regions. This file is then made available to you through this website, updated daily so you always have the latest information.

As a bonus feature, if you stay on this page for more than a minute, you will be automatically redirected to the Azure Regions Data page.
